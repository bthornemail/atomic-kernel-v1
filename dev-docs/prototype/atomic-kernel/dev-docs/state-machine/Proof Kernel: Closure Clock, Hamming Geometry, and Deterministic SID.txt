# Proof Kernel: Closure Clock, Hamming Geometry, and Deterministic SID

## Goal

Define the **smallest deterministic system** that:

1. Starts from a seed state.
2. Expands deterministically inside bounds.
3. Produces a canonical closed form.
4. Projects that form to a phase class (`mod 7`).
5. Generates a stable semantic identifier (**SID**).

All higher interpretations (Unicode, geometry metaphors, etc.) are **optional layers** above this kernel.

---

# 1. Base Space

Define the byte space:

```text
B = {0,1}⁸
```

Each element is an 8-bit word.

Example:

```
00000000
01010110
11111111
```

This space has **256 states**.

---

# 2. Metric Structure

Define **Hamming distance**:

```text
d(x,y) = number of differing bit positions
```

Algorithm:

```python
def hamming_distance(x: int, y: int) -> int:
    return (x ^ y).bit_count()
```

Properties:

* `d(x,y) ≥ 0`
* `d(x,y) = 0 ↔ x=y`
* `d(x,y) = d(y,x)`
* triangle inequality holds

Therefore `(B, d)` is a **metric space**.

---

# 3. Adjacency Graph

Define adjacency:

```text
x ~ y  iff  d(x,y) = 1
```

This means two states differ by exactly one bit.

Algorithm:

```python
def neighbors(x: int):
    for i in range(8):
        yield x ^ (1 << i)
```

This creates the **8-dimensional hypercube graph**.

Each node has exactly **8 neighbors**.

---

# 4. Bounded Expansion

Define three sets:

```
seed  S
lower bound L
upper bound U
```

where:

```
L ⊆ U ⊆ B
```

Expansion step:

```text
E(X) = (X ∪ neighbors(X) ∪ L) ∩ U
```

Algorithm:

```python
def expand(X, lower, upper):
    Y = set(X)
    for x in X:
        for n in neighbors(x):
            if n in upper:
                Y.add(n)
    Y |= lower
    Y &= upper
    return Y
```

---

# 5. Closure Operator

Define closure as the **least fixed point** of expansion.

Algorithm:

```python
def closure(seed, lower, upper):
    X = set(seed)
    while True:
        Y = expand(X, lower, upper)
        if Y == X:
            return X
        X = Y
```

### Proof: closure properties

1. **Extensive**

```
seed ⊆ closure(seed)
```

because expansion only adds elements.

2. **Monotone**

```
A ⊆ B ⇒ cl(A) ⊆ cl(B)
```

because expansion rules are identical.

3. **Idempotent**

```
cl(cl(X)) = cl(X)
```

because the algorithm stops exactly at a fixed point.

Therefore `closure()` is a valid **closure operator**.

---

# 6. Phase Projection

Define projection:

```text
π(X) = |X| mod 7
```

or for a single byte:

```text
π(x) = popcount(x) mod 7
```

Algorithm:

```python
def project_mod7(X):
    return len(X) % 7
```

This maps any closed state to one of:

```
{0,1,2,3,4,5,6}
```

These become **phase states**.

---

# 7. Canonical Representation

Closed states must have a deterministic encoding.

Algorithm:

```python
def canonical_state(X):
    return tuple(sorted(X))
```

This guarantees the same set always produces the same representation.

---

# 8. Semantic Identity (SID)

Define SID:

```text
SID = SHA256(canonical_state)
```

Algorithm:

```python
import hashlib

def sid_of_state(X):
    data = bytes(sorted(X))
    return hashlib.sha256(data).hexdigest()
```

### Identity Law

If two seeds produce the same closed state:

```
closure(seed₁) = closure(seed₂)
```

then:

```
SID(seed₁) = SID(seed₂)
```

Therefore SID represents the **identity of the canonical closure**, not the seed itself.

---

# 9. Deterministic Clock Step

Define a deterministic seed transition.

Example:

```python
def advance_seed(seed):
    return {x ^ 1 for x in seed}
```

Clock step:

```python
def step(seed, lower, upper):
    closed = closure(seed, lower, upper)
    tick = project_mod7(closed)
    sid = sid_of_state(closed)
    next_seed = advance_seed(seed)
    return next_seed, tick, sid
```

This produces a deterministic sequence:

```
(seed₀ → seed₁ → seed₂ …)
```

with corresponding:

```
closure → phase → SID
```

# 10. Relation to Coding Theory

In **7-bit space**, the Hamming(7,4) code uses a parity-check matrix whose columns correspond to the **7 points of the Fano plane**.

A triple `{i,j,k}` forms a line iff:

```
ci ⊕ cj ⊕ ck = 0
```

This gives the well-known structure:

```
7 points
7 lines
3 points per line
```

Thus the `%7` phase projection naturally aligns with the **7-point combinatorial structure** used in error-correcting codes.

The minimal system consists of:

```
SID = SHA256(canonical)   semantic identity
step()                    deterministic evolution
```

Everything else (Unicode layout, symbolic meaning, etc.) can be built on top.

## 1. You’re right about representation

The kernel mixed two representations:

* **set-of-bytes**
* **bitmask/ring operations**

That was useful for exposition, but not good for a real spec.

For implementation and geometry, the cleaner choice is:

```text
primary representation = bitmask
```

because it preserves:

* locality
* closure-as-expansion
* explicit bounds via `& upper`
* explicit seed growth on the cyclic ring

So if we tighten the kernel, Section 4 and 5 should be rewritten in bitmask form.

A cleaner 7-bit version is:

```python
MASK7 = 0x7f

def expand7(x: int) -> int:
    left  = ((x << 1) | (x >> 6)) & MASK7
    right = ((x >> 1) | ((x & 1) << 6)) & MASK7
    return x | left | right

def closure7(seed: int, lower: int, upper: int) -> int:
    x = (seed | lower) & upper
    while True:
        y = expand7(x) & upper
        y = (y | lower) & upper
        if y == x:
            return x
        x = y
```

That version keeps the geometry visible.

## 4. You’re right about the termination bound

That should have been stated explicitly.

The closure algorithm terminates because:

* the space is finite
* expansion is monotone
* no element is ever removed

So in the set-based 8-bit version:

```text
closure() terminates in at most |U| steps
```

because each strict iteration adds at least one new point and there are at most `|U|` points available.

In the 7-bit mask version, the bound is even tighter:

```text
closure7() terminates in at most popcount(upper) steps
```

since each strict iteration turns on at least one previously off bit inside `upper`.

That’s a better statement.
